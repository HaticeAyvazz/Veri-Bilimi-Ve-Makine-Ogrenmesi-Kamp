# 🍷 Wine Quality EDA

Bu proje, Kaggle’daki **Wine Quality** veri seti üzerinde **interaktif keşifsel veri analizi (EDA)** yapmanızı sağlayan bir **Streamlit web uygulamasıdır**.  

---

## 🚀 Özellikler

- **Renk Paleti Seçimi:** Kullanıcı, sidebar üzerinden farklı renk paletleri seçerek grafiklerin görünümünü değiştirebilir.
- **Interaktif Grafikler:**  
- **Interaktif Sidebar:** Kullanıcı, hangi grafikleri görmek istediğini seçebilir ve anında görselleştirmeyi güncelleyebilir.
- **Streamlit Arayüzü:** Kolay kullanım ve anlık güncellenen görselleştirmeler.  

---

## 🛠 Teknolojiler

- Python, **Pandas**, **Matplotlib**, **Seaborn**, **Streamlit**
